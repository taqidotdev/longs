# Longs

A simple voice recording app, made for [remixed](https://remixed.hackclub.com/) by Hack Club!

Made using Next.js, Supabase, and Tone.js

Check it out at https://longs.taqi.dev/ !!

## Features

* Record voice notes
* Add notes to them
* Add effects:
  * Reverb
  * Ping Pong Delay
  * Pitch Shift
  * Distortion
  * Gain
